
Extensions
==========

This describes optional features that are not part of baseline.
Extensions should have published working code, and be supported in more
than one OpenRaster-aware application.

   :maxdepth: 1
   :caption: Currently available extensions are:

   Layer Edit Locking Status — load & save layer immutability state <layer-edit-locking-status>
